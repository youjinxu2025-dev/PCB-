# PCB审图工坊

这是一个面向电子工程师的在线接单网站，支持：

- 用户注册 / 登录
- 上传原理图、PCB、Gerber、BOM 等资料
- 自动生成订单
- 支付登记
- 后台查看订单和付款状态

## 本地开发

1. 安装 Node.js 20 或更高版本
2. 在项目目录运行：

```powershell
npm start
```

3. 打开浏览器访问：

```text
http://127.0.0.1:3000
```

## 发布部署

项目已经补齐以下发布能力：

## 当前适用场景

这版适合：

- 先快速发布测试
- 先验证线上接单流程
- 再逐步升级数据库、对象存储和真实支付
